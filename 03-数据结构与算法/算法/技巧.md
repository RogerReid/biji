# 交换整型数的三种方法
[[交换整型数]]
## 一，设置中间变量
```C
void swap(int *p1,int *p2) 
{ int temp;
 
  temp=*p1;
   
  *p1=*p2;
   
  *p2=temp;
}
```

## 二，加减法
**加减法的弊端**：容易产生溢出，如果两个数都特别大的话，那么就容易溢出。
```C
#include <stdio.h>
int main()
{
	int a=1, b=2;
	
	a = a+b;
	
	b = a-b;
	
	a = a-b;
	
	printf("a=%d,b=%d", a,b);  // a=2, b=1;
}
```

## 三，用异或交换
该方法不会出现溢出的问题。
```C
int main()

{

    int a=1, b=2;

    a = a^b;

    b = a^b;

    a = b^a;

    printf("a=%d, b=%d", a, b);  // a=2,b=1;

}
```
