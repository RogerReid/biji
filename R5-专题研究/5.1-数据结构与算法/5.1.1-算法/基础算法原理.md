# 基础
## 位运算
### lowbits原理
- **作用** 找到某数n的最后一位二进制1。
- **(x  & - x)** 返回的只有最后一位是1，其他位都是0。 
0 - 原码 = 补码
所以  - 源码 = ~ 原码 + 1   （~取反标志）
( -x = ~x + 1)

### 输出n的每一位二进制数
- 与上1就是看二进制的最后一位是什么
```c++
int main()
{
	int n;
	cin >> n;
	for(int k = 3; k >=0; k --)
	{
		
		cout << (n >> k & 1) << endl;
		
	}
}
```
